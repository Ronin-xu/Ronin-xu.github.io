1. 什么是Python 解释器?

  • Python 解释器是运行你所编写的 Python 程序的程序。

2. 什么是源代码?

  • 源代码是为程序所写的语句:它由文本文件中的文本组成，通常以 .py 作为后缀名。

3. 什么是字节码?

  • 字节码是Python将程序编译后所得到的低级形式。Python自动将字节码保存到后缀名为.pyc 的文件中。

4. 什是 PVM?

  • PVM是Python 虚拟机，它是Python的运行时引擎，能解释编译得到的字节码。

5. CPython Jython 以及 lronPython 有什么不同?

  • CPython是Python 语言的标准实现。 Jython IronPython 分别是 Python程序在Java
    .NET 环境中的实现，它们都是 Python 的替代编译器。

6. 什么是 Stackless PyPy?

  • Stackless是一种针对井发性而增强的 Python 版本，而 PyPy 是针对速度而增强的
    Python 实现。 PyPy 作为 Psyco 的继任者，融合了 Psyco 中的 JIT 概念。
